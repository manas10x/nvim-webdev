# nvim-webdev

A fast and minimal Neovim configuration tailored for web development with HTML, CSS, JavaScript, and modern workflow tools.

## Features

- Modern web development workflow (HTML, CSS, JS)
- Treesitter for syntax highlighting
- LSP support for smart code completion
- Autopairs, snippets, and Emmet support
- Telescope for fuzzy file search
- Harpoon for quick navigation

## Installation

1. Backup your existing Neovim config:
   ```bash
   mv ~/.config/nvim ~/.config/nvim.backup

2. Install required system packages:
   ```bash
   npm install -g prettier
   npm install ‑g emmet‑ls
